Pretendard Font Installation Guide
===================================

Download Pretendard fonts from:
https://github.com/orioncactus/pretendard/releases

Download "Pretendard-1.3.9.zip" and extract the following OTF files
from "public/static" folder to this directory:

- Pretendard-Regular.otf
- Pretendard-Medium.otf
- Pretendard-SemiBold.otf
- Pretendard-Bold.otf
